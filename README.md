
Pyton simple calculator
      A lightweight command-line tool to perform basic arithmetic operations.
Features include
- Operations: Addition, Subtraction, Multiplication, and Division.
- Safety: Built-in error handling to prevent crashes from division by zero.
- CLI Interface: Easy-to-use command-line prompts.

How to run
     1. Ensure you have Python installed.
     2. Clone this repository or download calculator.py.
     3. Run the script using: python calculator.py
